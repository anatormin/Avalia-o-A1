# Exercícios de Python: Estruturas Condicionais, Repetição, Listas e Dicionários
Este repositório contém quatro exercícios práticos desenvolvidos para o módulo de introdução à lógica de programação em Python.

- Estruturas Condicionais

Arquivo: Estruturas_Condicionais.ipynb

Este programa simula um sistema de verificação de idade para entrada em eventos.
Ele solicita ao usuário a idade e utiliza estruturas condicionais (if, elif, else) para determinar se a pessoa pode entrar, se precisa de autorização ou se a entrada não é permitida.
A lógica avalia diferentes faixas etárias e apresenta mensagens adequadas.
O exercício trabalha entrada de dados, tomada de decisão e organização clara das condições.

- Estruturas de Repetição

Arquivo:  Estruturas_de_Repetição.ipynb

Realizam a mesma tarefa utilizando laços distintos: um com for e outro com while.
Eles percorrem uma sequência numérica de 1 a 100 e exibem apenas os números pares dentro desse intervalo.
O objetivo do exercício é demonstrar como diferentes estruturas de repetição podem alcançar o mesmo resultado, reforçando o uso de contadores, condições dentro do loop, operadores lógicos e organização das etapas do fluxo.

- Listas

Arquivo: Lista.ipynb

Este programa permite ao usuário inserir nomes de alunos de forma dinâmica.
Cada nome digitado é armazenado em uma lista, e o processo continua até que um comando de parada seja informado.
Ao final, o programa exibe todos os nomes coletados, destacando como listas funcionam como estruturas de armazenamento flexíveis.
O exercício trabalha entrada contínua de dados, uso de listas, manipulação de coleções e impressão organizada dos elementos armazenados.

- Dicionários

Arquivo: Dicionários.ipynb

Este programa implementa um sistema simples de cadastro de produtos, onde cada produto possui nome e preço.
Os dados são armazenados em um dicionário, usando o nome como chave e o preço como valor.
O usuário pode adicionar quantos produtos desejar, e ao final o programa exibe todos os itens cadastrados de maneira clara.
O exercício reforça o uso correto de dicionários, inserção e recuperação de dados, além da construção de estruturas de armazenamento mais estruturadas e funcionais que permitem associações entre informações.
